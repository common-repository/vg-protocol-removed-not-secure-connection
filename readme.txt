== Description ==
VG Protocol Removed:Not Secure Connection is very simple and low weight plugin which helps WordPress/dashboard user to remove protocol from website link.

When we switch the protocol from HTTP to HTTPS then we have so some URL/files are run in HTTP protocol and it will show the warning message and show your site is not secure.

== Installation ==
*	Download the plugin .zip file.
*	Login in to admin Click Plugins -> Add New -> Upload.
*	Find VG Protocol Removed:Not Secure Connection in plugin list  and activate it.
